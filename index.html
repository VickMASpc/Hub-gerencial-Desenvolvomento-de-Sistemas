Projeto 1 — Sistema de pontos

Beta funcional

Sistema simples para registro de presença de funcionários. Permite que administradores criem departamentos, cargos e funcionários, enquanto os funcionários podem bater ponto e consultar suas informações e entradas registradas.

Funcionalidades principais

  • Cadastro de departamentos.
  • Cadastro de cargos.
  • Cadastro e gerenciamento de funcionários.
  • Registro de entrada e saída de ponto.
  • Consulta de informações pessoais do funcionário.
  • Consulta de entradas registradas.

Ação necessária

Refatoração necessária
  • O sistema deve ser refatorado seguindo a questão dos QR codes discutida no GitHub.

Projeto 2 — E-commerce, logística e vendas

Beta funcional / ecossistema integrado

Projeto composto por sistemas conectados para operação comercial: website de vendas, sistema de logística e sistema de análise de vendas em estágio inicial.

Sistema integrado

Website de vendas

Website de e-commerce com listagem de produtos, carrinho funcional e checkout conectado ao sistema local.

Itens especificados

  • Listagem de produtos.
  • Carrinho funcional.
  • Checkout conectado ao sistema local.
  • Atualização automática dos produtos cadastrados no sistema.
Sistema interno

Sistema de logística

Sistema de gestão de estoque voltado ao controle de produtos, pedidos recebidos e envio.

Itens especificados

  • Cadastrar produtos.
  • Remover produtos.
  • Editar quantidades em estoque.
  • Acompanhar produtos com baixo estoque.
  • Integrar dados com pedidos recebidos pelo e-commerce.
  • Dar baixa nos produtos para envio dos pedidos.
Alpha

Sistema de análise de vendas

Sistema em desenvolvimento para que a equipe de vendas possa ter tracking de vendas e insights.

Itens especificados

  • Acompanhamento de vendas.
  • Geração de insights para a equipe comercial.
  • Base inicial para futuras métricas, relatórios e dashboards.
