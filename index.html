Projeto 1 — Sistema de Controle de Ponto por QR Code

MVP QR funcional

Sistema de presença por QR Code para ambientes escolares. Administradores gerenciam departamentos, cargos, funcionários e gestores; gestores registram entradas e saídas ao escanear os crachás pela câmera do dispositivo; e funcionários consultam seu próprio histórico.

Funcionalidades principais

  • Painel administrativo para departamentos, cargos, funcionários e gestores.
  • Geração automática de QR Code único e impressão de crachá.
  • Scanner de QR Code via câmera do navegador para o gestor do departamento.
  • Confirmação de entrada ou saída após a leitura do crachá.
  • Visualização de presentes do dia e scans recentes para gestores.
  • Consulta de histórico de ponto pelo funcionário.
  • Registro do método camera-qr para rastreabilidade da presença.

Papéis de acesso

  • Admin: gestão completa e visualização dos registros.
  • Gestor: scanner QR e acompanhamento do próprio departamento.
  • Funcionário: registro próprio e consulta de histórico.
  • TI: criação de administradores por código de acesso.

Implementação QR disponibilizada

Branch principal informada
  • Leitura de QR Code usando câmera nativa no navegador.
  • Geração de crachás QR diretamente na lista de funcionários.
  • Feedback visual para confirmação, câmera negada e QR incompatível.
  • Para produção, é necessário publicar as regras Firestore, habilitar login por e-mail/senha e criar o índice composto de presença.

Projeto 2 — E-commerce, logística e vendas

Beta funcional / ecossistema integrado

Projeto composto por sistemas conectados para operação comercial: website de vendas, sistema de logística e sistema de análise de vendas em estágio inicial.

Sistema integrado

Website de vendas

Website de e-commerce com listagem de produtos, carrinho funcional e checkout conectado ao sistema local.

Itens especificados

  • Listagem de produtos.
  • Carrinho funcional.
  • Checkout conectado ao sistema local.
  • Atualização automática dos produtos cadastrados no sistema.
Sistema interno

Sistema de logística

Sistema de gestão de estoque voltado ao controle de produtos, pedidos recebidos e envio.

Itens especificados

  • Cadastrar produtos.
  • Remover produtos.
  • Editar quantidades em estoque.
  • Acompanhar produtos com baixo estoque.
  • Integrar dados com pedidos recebidos pelo e-commerce.
  • Dar baixa nos produtos para envio dos pedidos.
Alpha

Sistema de análise de vendas

Sistema em desenvolvimento para que a equipe de vendas possa ter tracking de vendas e insights.

Itens especificados

  • Acompanhamento de vendas.
  • Geração de insights para a equipe comercial.
  • Base inicial para futuras métricas, relatórios e dashboards.
